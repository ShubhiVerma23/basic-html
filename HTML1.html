What is HTML?

HTML (HyperText Markup Language) is the standard markup language used to structure and define content: like text, images, links, and forms on web pages.

Elements in HTML

  • Header
  • Paragraph
  • Buttons
  • Division and many more

What is CSS?

CSS (Cascading Style Sheets) is a style sheet language that defines how HTML elements should be displayed, controlling layout, colors, fonts, and more.

Key Features of CSS

  • Separation of content and presentation
  • Reusable styles across multiple pages
  • Cascading rules that determine priority
  • Supports responsive design via media queries
  • Makes web pages visually appealing

Example of CSS Usage

    h1 {
      color: blue;
      font-size: 2em;
    }

This rule would make all <h1> headings appear in blue and twice the default size.

Why Use CSS?

CSS saves time and effort by letting you style multiple pages in one file, improves maintenance, and enhances user experience via better layout and design.

What is JavaScript?

JavaScript is a dynamic, interpreted programming language that enables interactivity on web pages—such as animations, event handling, and dynamic content updates—running in browsers and beyond (e.g. Node.js).

Main Features

  • Client-side scripting: runs in the browser for interactive behavior (e.g. drop-down menus) .
  • Server-side usage: supports non-blocking, event-driven apps with Node.js.
  • Multi-paradigm: supports imperative, functional, and prototype-based object‑oriented styles.
  • Asynchronous programming: uses callbacks, promises, and async/await to handle background tasks.

Example Code

<!DOCTYPE html>
<html>
<head><title>JS Example</title></head>
<body>
  <p id="demo">Click the button</p>
  <button onclick="document.getElementById('demo').innerHTML = 'Hello JavaScript!'"> Do it</button>
</body>
</html>

Why JavaScript?

JavaScript brings websites to life—enabling rich user interfaces, server-side applications, mobile apps, games, and more.
